# R10

An application to to find the details of the upcoming Developer Conference called R10. 

## Installation

Download the repo, then run:

```bash
npm install
```

## Start-up

From inside the `client` directory, run:

```bash
react-native run-ios --simulator "iPhone SE".
```

## Technology Used
* React-Native
* Redux
* Git - Github
